# BUPT-elective-course

本榜由广大同学提供数据，陈艾斯整理形成，我会努力收集大家数据并持续更新，也欢迎大家持续为我提供最新数据与消息。

针对过去红黑榜课程换老师、主观性太强等问题制作此榜，本榜不列（红、黑）客观列出同学得分情况和上课情况供大家综合性考虑，避免主观造成错误。

本榜单与2026年1月21日制成，考核方式可供参考，有些课考核方式会变，如果变了，请大家联系我修改。
